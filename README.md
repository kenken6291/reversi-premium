# Reversi Premium (オセロゲーム)

高品質なビジュアルデザインと3段階のAIを搭載した、ブラウザで動作するプレミアムなオセロ（リバーシ）ゲームです。

## 🌟 特徴
- **マルチ対戦モード**: 1台の端末で交互に打つ「ローカル2人対戦」と、「VS コンピュータ（AI戦）」をサポート。
- **3段階のAI難易度**: 
  - **簡単**: 完全ランダム打ち
  - **普通**: 盤面評価（角の優先度など）による思考
  - **難しい**: ミニマックス法（Minimax）にアルファベータ（Alpha-Beta）枝刈りを組み合わせ、数手先を読んで最適解を導き出すAI。
- **プレミアムデザイン**: ガラスモフィズム（透過ガラススタイル）、3Dディスク反転アニメーション、ダークモード等を含む4種のテーマ（クラシックグリーン、ダークガラス、サイバーネオン、フォレストウッド）を搭載。
- **インタラクティブサウンド**: Web Audio APIを使用し、ゲーム内で動的に合成した効果音（石の打音、反転音、勝敗ファンファーレ）を再生。外部音声ファイルのロードが不要です。
- **便利機能**: 
  - 1手戻す機能（Undo）
  - ローカルストレージを使用した累計戦績の保存
  - 終局後の「棋譜再生」機能（一手ずつ進めたり、自動再生して振り返ることができます）

## 🚀 起動方法
本アプリケーションはHTML、CSS、JavaScriptのみで構築されているため、特別なインストール作業は不要です。

1. このフォルダ内の `index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップします。
2. もしくは、ローカルサーバー（VS CodeのLive Serverなど）を起動してアクセスします。

## 🛠️ GitHubへの登録とインターネット公開手順

このゲームをGitHubにアップロード（登録）し、さらに誰でも遊べるようにインターネット上に公開（GitHub Pages）する手順です。

### 1. ローカルでの準備
すでにこのフォルダで Git は初期化され、最初のコミットが作成されています。

### 2. GitHub上で新しいリポジトリを作成
1. [GitHub](https://github.com/) にログインします。
2. 画面右上の「**+**」アイコンをクリックし、「**New repository**」を選択します。
3. **Repository name**（例: `reversi-premium`）を入力します。
4. 説明（Description）に「HTML5 Reversi Game」などと入力します。
5. 公開範囲を「**Public**（パブリック）」に設定します。（インターネット公開に必要です）
6. 「Initialize this repository with:」の項目はすべて**チェックを入れずに空欄のまま**にします。
7. 「**Create repository**」ボタンをクリックします。

### 3. リポジトリをGitHubに接続してプッシュする
GitHubリポジトリ作成後に表示される「…or push an existing repository from the command line」のコマンドをターミナル（PowerShellやBashなど）で実行します。

```bash
# 1. ブランチ名を main に設定
git branch -M main

# 2. リモートリポジトリ（GitHub）の登録
# ※URLの "username" と "reversi-premium" はご自身の環境に合わせて変更してください
git remote add origin https://github.com/ユーザー名/reversi-premium.git

# 3. GitHubへソースコードをアップロード
git push -u origin main
```

> [!NOTE]
> `git push` 時にサインインを求められた場合は、ブラウザでのサインイン認証またはパーソナルアクセストークンを入力してください。

### 4. インターネット上に無料公開する（GitHub Pages）
GitHubにプッシュしたら、次の手順でゲームをウェブサイトとして公開できます。

1. GitHubのリポジトリページにアクセスします。
2. 上部タブの「**Settings**（設定）」をクリックします。
3. 左メニューの「Code and automation」カテゴリにある「**Pages**」をクリックします。
4. 「Build and deployment」の中の **Source** を「**Deploy from a branch**」にします。
5. **Branch** で `main`（または `/ (root)`）を選択し、「**Save**」をクリックします。
6. 数分待つと、画面上部に公開されたURL（例: `https://ユーザー名.github.io/reversi-premium/`）が表示されます。このURLにアクセスすれば、スマホや他のPCからいつでもこのゲームをプレイできます！
